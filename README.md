# Проект для 3-МД-21

## Дисциплина: Современные технологии разработки

Учебный проект для практических занятий по веб-разработке.

## Структура курса

### Семестр 1 — Верстка (HTML/CSS)

- **Уроки** — материалы по HTML, CSS, Grid, Flexbox, адаптивной верстке
- **Задания** — практические работы по созданию сайтов

### Семестр 2 — JavaScript

- **Уроки** — основы JavaScript, функции, DOM, ООП, модули, API, асинхронность
- **Задания** — портфолио с JS, дашборд с виджетами, игра-платформер
- **Экзамен** — билеты и темы финального проекта

## Навигация

[index.html](index.html) — главная страница с выбором семестра и раздела (уроки, задания, экзамен).

## Структура файлов

```
├── index.html              # Главная — учебные материалы
├── index.json               # Индекс материалов (генерируется)
├── generate-index.js        # Скрипт генерации индекса
├── assets/
│   ├── semester-1/          # Семестр 1 — Верстка
│   │   ├── lessons/         # Уроки
│   │   ├── assignments/     # Задания
│   │   └── exam/            # Билеты
│   └── semester-2/          # Семестр 2 — JavaScript
│       ├── lessons/         # Уроки
│       ├── assignments/     # Задания
│       └── exam/            # Билеты и финальный проект
├── css/
├── js/
└── README.md
```

## Обновление индекса материалов

После добавления или изменения файлов в `assets/` выполните:

```bash
node generate-index.js
```

## Использование

1. **На занятиях** — разбор и модификация кода проекта
2. **Самостоятельно** — изучение материалов через [materials.html](materials.html)
3. **Git** — история изменений как справочный материал

> **Важно:** Содержимое проекта обновляется в процессе обучения.
