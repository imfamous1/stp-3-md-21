# Проект для 3-МД-21

## Дисциплина: Современные технологии разработки

Учебный проект для практических занятий по веб-разработке. Материалы организованы по семестрам: верстка (HTML/CSS) и JavaScript.

## Навигация

**[index.html](index.html)** — главная страница с выбором семестра и раздела (уроки, задания, экзамен).

Навигация: **Семестр** → **Раздел** (Уроки / Задания / Экзамен) → **Материал**

## Структура курса

### Семестр 1 — Верстка (HTML/CSS)

- **Уроки** — 14 уроков: от введения в HTML до Git
- **Задания** — сайт-визитка, сайт для бизнеса, лендинг по референсу
- **Экзамен** — 17 билетов по верстке

### Семестр 2 — JavaScript

- **Уроки** — 7 уроков: основы, функции, DOM, ООП, модули, API, формы
- **Задания** — портфолио-резюме, дашборд с виджетами, игра-платформер
- **Экзамен** — 20 билетов и темы финального проекта
- **Практикум** — дополнительный материал (docx) в разделе «Уроки»

## Возможности

- Выбор семестра и раздела
- Ссылки на видеолекции RUTUBE (внутри каждого семестра)
- Скачивание практикума по JavaScript
- Кнопка «Поделиться» для отправки ссылки
- Переключение светлой/тёмной темы
- Адаптивный дизайн в стиле iOS (mobile-first)

├── assets/

│   ├── semester-1/         # Семестр 1 — Верстка
│   │   ├── lessons/        # 14 уроков
│   │   ├── assignments/    # 3 задания
│   │   └── exam/          # Билеты

## Обновление индекса

После добавления или изменения файлов в `assets/` выполните:

```bash
node generate-index.js
```

## Использование

1. **На занятиях** — разбор и модификация кода проекта
2. **Самостоятельно** — изучение материалов через главную страницу
3. **Git** — история изменений как справочный материал

> **Важно:** Содержимое проекта обновляется в процессе обучения.
