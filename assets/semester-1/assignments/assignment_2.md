# Задание №2: Создание сайта-визитки для малого бизнеса

**Автор:** Евгений Кокорин  
**Дата:** November 04, 2025

## Цель

Разработать полнофункциональный лендинг для малого бизнеса или личного проекта, сочетающий современный дизайн с практической пользой.

## Технические требования

- Чистый HTML/CSS + минимальный JavaScript для интерактивности

### Обязательное использование:

- **CSS Grid** и/или **Flexbox** для основной структуры
- **Адаптивная верстка** (мобильные устройства + десктоп)
- **Семантическая разметка HTML5**

### JavaScript для:

- Плавной прокрутки по якорным ссылкам
- Модального окна
- Валидации формы

## Содержание сайта

### Шапка (Header)

- Лого/название бизнеса
- Навигационное меню с якорными ссылками

### Главная секция (Hero section)

- Крупный заголовок
- Описание проекта/услуги
- Кнопка-призыв к действию

### О проекте/Услуги

- Карточки услуг/товаров с описанием

### Галерея/Портфолио

- Изображения работ/продуктов

### Контакты

- Форма обратной связи
- Контактная информация (телефон, email, соцсети)

### Подвал (Footer)

- Копирайт
- Повтор основных контактов
