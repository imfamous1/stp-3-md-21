# Задание №3: Вёрстка лендинга по референсу с анимациями

**Автор:** Евгений Кокорин  
**Дата:** November 04, 2025

## Цель

Развить навык переноса визуального дизайна в рабочий код. Научиться анализировать существующий сайт, разбивать его на структурные блоки и точно воссоздавать его с помощью HTML, CSS и JavaScript, включая интерактивные элементы и анимации.

## Технические требования

- Чистый HTML/CSS/JavaScript

### Структура и сетка

- Использование **CSS Grid** и/или **Flexbox** для точного соответствия макету

### Адаптивность

- Сайт должен корректно отображаться на основных разрешениях (мобильные устройства, планшеты, десктоп)
- Референс можно адаптировать, если его исходная версия не адаптивна

### Семантическая разметка

- **Семантическая разметка HTML5**

### CSS Анимации и переходы

- Обязательно воспроизвести все основные анимации и эффекты наведения (hover), плавные переходы, которые есть на референсе

## Процесс работы

### Выбор референса и анализ

Выберите один лендинг для анализа. Рекомендуемые источники:

- **Tilda Made on Tilda:** https://tilda.cc/ru/madeontilda/ (выбирайте не самые сложные проекты, обратите внимание на сайты услуг, портфолио, стартапов)
- **Awwwards:** https://www.awwwards.com/ (для вдохновения, выбирайте сайты с умеренной сложностью)
- Любой другой понравившийся лендинг (главное – наличие интерактива и анимаций)

В таблице отчетности вашего проекта обязательно добавьте ссылку на референс и скриншоты (общий вид и ключевые элементы).

### Планирование и декомпозиция

- Визуально разбейте сайт на основные блоки (Header, Hero, About, Features, Testimonials, Footer и т.д.)
- Проанализируйте сетку макета. Сколько колонок? Как расположены элементы?
- Выпишите цветовую палитру, шрифты
- Составьте список всех анимаций и интерактивных элементов, которые нужно будет реализовать

### Использование нейросетей для помощи

- **Для анализа:** Загрузите скриншот сложного блока в нейросеть и попросите предложить структуру HTML и основные свойства CSS для его реализации
- **Для кода:** Можете использовать нейросеть для генерации базовой структуры или подсказок по реализации конкретной анимации на CSS/JS
- **Для обучения:** Посмотрите видео-гайд по верстке макетов с помощью нейросети: Rutube https://rutube.ru/plst/877516/ (видео 2, 3, 5). Это поможет понять рабочий процесс
