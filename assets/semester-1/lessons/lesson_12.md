# Формы

Тег `<form>` - контейнер для группы элементов управления (полей ввода, кнопок), который собирает данные от пользователя для отправки на сервер.

## Ключевые атрибуты

- **`action`**: URL-адрес, на который отправляются данные формы, например `action="/submit-form"`
- **`method`**: HTTP-метод для отправки данных (`GET` или `POST`)

Пример:

```html
<form action="/submit-form" method="POST">
    <!-- поля формы -->
</form>
```

## Элементы форм

### `<input>`

Универсальный элемент для ввода данных.

**Атрибут `type`**: определяет поведение поля

- `text` - однострочный текст
- `email` - для email с базовой валидацией
- `password` - скрывает ввод символов
- `number` - для числового ввода
- `checkbox` - множественный выбор
- `radio` - одиночный выбор из группы
- `date` - выбор даты
- `file` - загрузка файла

**Другие важные атрибуты:**

- `name` - обязателен для отправки данных, это ключ, по которому сервер получит значение
- `placeholder` - подсказка внутри поля
- `required` - значение обязательное (атрибут без значения)

Пример:

```html
<input type="text" name="username" placeholder="Введите имя" required>
<input type="email" name="email" placeholder="Введите email" required>
```

### `<textarea>`

Многострочное текстовое поле.

Атрибуты `rows`/`cols` - для базовых размеров, но лучше задавать через CSS.

```html
<textarea name="message" rows="5" cols="30" placeholder="Введите сообщение"></textarea>
```

### `<select>`

Раскрывающийся список для выбора, внутри содержит теги `<option>` для каждого пункта.

```html
<select name="country">
    <option value="ru">Россия</option>
    <option value="us">США</option>
    <option value="uk">Великобритания</option>
</select>
```

### `<button>`

Кнопка. Важно указывать атрибут `type`:

- `type="submit"` - отправляет форму по умолчанию
- `type="button"` - простая кнопка без действия по умолчанию
- `type="reset"` - сбрасывает форму

```html
<button type="submit">Отправить</button>
<button type="button">Обычная кнопка</button>
```
