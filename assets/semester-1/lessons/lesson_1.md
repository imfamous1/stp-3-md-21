# Введение в HTML. Структура документа

HTML (HyperText Markup Language) — язык разметки, который описывает структуру веб-страницы. Браузер читает HTML-код и отображает его как страницу с текстом, изображениями и ссылками.

## Минимальная структура документа

Каждая HTML-страница начинается с объявления типа документа и базовой структуры:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
</head>
<body>
    Содержимое страницы
</body>
</html>
```

## Разбор по частям

### `<!DOCTYPE html>`
Сообщает браузеру, что это HTML5-документ. Обязательная первая строка.

### `<html lang="ru">`
Корневой элемент. Атрибут `lang` указывает язык страницы (для поисковиков и скринридеров).

### `<head>`
Служебная часть: метаданные, заголовок, подключение стилей и скриптов. Не отображается на странице.

### `<meta charset="UTF-8">`
Кодировка символов. UTF-8 нужна для корректного отображения кириллицы.

### `<meta name="viewport" ...>`
Настройки отображения на мобильных устройствах.

### `<title>`
Заголовок вкладки браузера и в результатах поиска.

### `<body>`
Видимое содержимое страницы: текст, картинки, кнопки и т.д.

## Теги и элементы

HTML состоит из **тегов**. Большинство тегов парные: открывающий `<тег>` и закрывающий `</тег>`.

```html
<p>Это параграф</p>
```

Некоторые теги одиночные (самозакрывающиеся):

```html
<img src="photo.jpg" alt="Фото">
<br>
<input type="text">
```

## Атрибуты

Теги могут иметь атрибуты — дополнительные параметры:

```html
<a href="https://example.com">Ссылка</a>
<img src="image.png" alt="Описание">
```
