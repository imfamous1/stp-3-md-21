# Введение в CSS. Селекторы и каскад

CSS - **Cascade Style Sheet** (каскадная таблица стилей)

```css
селектор {
    свойство: значение;
    свойство: значение;
}
```

- **Селектор** - это таргет, который выбирает HTML-элемент для стилизации.
- **Свойство** - аспект внешнего вида, который мы хотим изменить (`color`, `font-size`).
- **Значение** - конкретная настройка для свойства (например, `red`, `20px`).

Пример:

```css
h1 {
    color: darkblue;
    font-size: 32px;
    text-align: center;
}
```

Этим правилом мы говорим браузеру: все заголовки `<h1>` сделай темно-синими, размером шрифта в 32px, а также выровняй по центру.

## Способы подключения CSS к HTML

### 1. Внешняя таблица стилей

Наилучший подход. Стили пишутся в отдельном файле с расширением `.css` и подключаются в `<head>` HTML-документа с помощью тега `<link>`.

```html
<link rel="stylesheet" href="style.css">
```

### 2. Внутренние стили

Через тег `<style>` в разделе `<head>` HTML-документа.

```html
<head>
    <style>
        h1 {
            color: red;
        }
    </style>
</head>
```

### 3. Встроенные стили (inline CSS)

Лучше не использовать. Здесь стили применяются напрямую к элементу через атрибут `style`.

```html
<h1 style="color: red">Кокорин Евгений</h1>
```

## Селекторы - инструмент для выбора элемента

### 1. Селектор по тегу

Выбираем все элементы с указанным тегом.

```css
p {
    color: blue; /* все <p> станут синими */
}
```

### 2. Селектор по классу

Самый популярный вариант - выбирает все элементы, у которых в атрибуте `class` есть указанное имя. Пишется через точку `.имя_класса`.

```css
.header__about {
    color: green;
}
```

### 3. Селектор по ID

Выбирает один элемент с указанным `id`. Пишется через `#`.

```css
#skills {
    background-color: aquamarine;
}
```

### 4. Универсальный селектор

Выбирает абсолютно все элементы на странице, часто используется для сброса стилей.

```css
* {
    margin: 0;
    padding: 0;
}
```

## Каскад и наследование

### Каскад

Каскад - механизм, который решает, какое именно CSS-правило будет применено к элементу, когда на него претендует несколько правил.

У нас учитывается несколько факторов:

#### 1. Важность (!important)

Объявление свойства с атрибутом `!important` имеет наивысший приоритет.

```css
h2 {
    color: darkcyan !important; 
}
```

#### 2. Специфичность

Это своего рода "вес" селектора, и у него есть своя формула, по которой определяется "победитель":

- Инлайновые стили - 1000 очков
- ID - 100 очков
- Классы - 10 очков
- Теги - 1 очко
- Универсальный селектор - 0 очков

#### 3. Порядок следования

Если специфичность одинакова, применяется правило, написанное позже.

```css
p { color: red; }
p { color: blue; }  /* будет этот цвет, т.к. он написан позже */
```

### Наследование

Некоторые CSS-свойства (например, `color`, `font-size`) автоматически наследуются от родительских элементов к дочерним.

Не наследуются, например: `margin`, `padding`, `width`, `height`.

Пример:

```html
<header>
    <h1>Кокорин Евгений</h1>
    <p>Преподаватель курса по веб-дизайну</p>
    <img src="images/avatar.jpg" alt="Портрет преподавателя Кокорина Евгения Сергеевича в деловом костюме" width="200" title="Это я!">
</header>
```

```css
header {
    color: red;
}
```

Текст внутри `<header>` будет красным.
