# JavaScript - Основы

## Введение

JavaScript — это:

- **Сценарный язык** (язык сценариев) для веб-страниц
- **JavaScript ≠ Java** — это разные языки программирования
- **Интерпретируемый язык** — код выполняется напрямую без предварительной компиляции

## IDE: Visual Studio Code / WebStorm / Cursor

Рекомендуемые редакторы кода для разработки на JavaScript.

## Подключение скриптов на веб-страницу

### Лучшие практики:

1. **Порядок загрузки**: скрипт подключается в конце `<body>`, чтобы не блокировать отрисовку HTML
2. **Внешние файлы (.js)** — предпочтительнее встроенного кода
3. **Переиспользование** на многих страницах
4. **Лаконичность кода** — разделение логики по файлам

### Пример подключения:

```html
<body>
  <!-- HTML контент -->
  <script src="js/main.js"></script>
</body>
```

## Синтаксис

### Основные правила:

1. **Точка с запятой (`;`)** — рекомендуется ставить для явного завершения инструкций
2. **Блоки кода**: `{}` — определяют область видимости
3. **Комментарии**:
   - `//` — однострочный комментарий
   - `/* */` — многострочный комментарий
4. **Интерполяция** (форматированная строка) — обратные кавычки `` ` ``

### Пример интерполяции:

```javascript
const name = "Иван";
const message = `Привет, ${name}!`; // "Привет, Иван!"
```

## Переменные

### Объявление переменных:

- `let` — переменная, значение которой можно изменить
- `const` — константа, значение нельзя изменить после объявления
- `var` — устаревшее, не рекомендуется использовать

### Правила именования:

1. **Стили именования**:
   - `snake_case` — `create_user_avatar`
   - `camelCase` — `createUserAvatar` (рекомендуется для JavaScript)

2. **Регистрозависимость**: `let A` и `let a` — две разные переменные

3. **Допустимые символы**: могут содержать символы `_`, `$`

4. **Ограничения**:
   - Не могут начинаться с цифры
   - Не могут совпадать с ключевыми словами (`if`, `for`, `function` и т.д.)

### Примеры:

```javascript
let userName = "Иван";
const MAX_SIZE = 100;
let _private = "скрыто";
let $element = document.querySelector('.item');
```

## Типы данных

### Примитивные типы:

1. **String** — строка текста
2. **Number** — число
3. **Boolean** — логическое значение (`true`/`false`)
4. **undefined** — значение не присвоено
5. **null** — пустое значение (намеренно установленное)

### Объекты:

- **Объекты** — коллекция свойств (ключ-значение)

### Проверка типа:

```javascript
const value = 42;
console.log(typeof value); // "number"
```

## Операторы

### Арифметические операторы:

- `+` — сложение
- `-` — вычитание
- `*` — умножение
- `/` — деление
- `%` — остаток от деления
- `**` — возведение в степень

### Операторы сравнения:

- `==` — проверка равенства (без учета типа)
- `===` — строгое равенство (с учетом типа) — **рекомендуется**
- `!=` — неравенство
- `!==` — строгое неравенство
- `>`, `<`, `>=`, `<=` — сравнение

### Инкрементация и декрементация:

- `++` — увеличить на 1
- `--` — уменьшить на 1

### Операторы присваивания:

- `+=` — `x += 5` эквивалентно `x = x + 5`
- `-=` — аналогично для вычитания

## Массивы

Массив — упорядоченный список данных.

### Примеры:

```javascript
let fruits = ['apple', 'orange', 'banana', 32, false];

// Обращение по индексу (индексация с 0)
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "banana"

// Массивы могут хранить элементы любых типов
let mixed = [1, "строка", true, null, { name: "Иван" }];
```

## Условные операторы: if...else / switch

### Структура if...else:

```javascript
let income = 100;

if (income > 100) {
    console.log("Доход больше 100");
} else if (income === 100) {
    console.log("Доход равен 100");
} else {
    console.log("Доход меньше 100");
}
```

### Структура switch:

```javascript
let day = "понедельник";

switch (day) {
    case "понедельник":
        console.log("Начало недели");
        break;
    case "пятница":
        console.log("Конец рабочей недели");
        break;
    default:
        console.log("Обычный день");
}
```

## Циклы

### Цикл while:

```javascript
let counter = 0;

while (counter < 10) {
    counter++; // counter = counter + 1
    console.log(counter);
}
```

### Цикл for:

```javascript
for (let i = 0; i < 10; i++) {
    console.log(i);
}
```

### Цикл for...of (для массивов):

```javascript
const fruits = ['apple', 'orange', 'banana'];

for (const fruit of fruits) {
    console.log(fruit);
}
```

## Преобразование типов

### Явное преобразование:

```javascript
let number = 42;
let string = "3.14";

let result1 = number + Number(string);
console.log(result1); // 45.14

// Другие способы явного преобразования
String(42);        // "42"
Boolean(1);        // true
parseInt("42");    // 42
parseFloat("3.14"); // 3.14
```

### Неявное преобразование:

JavaScript автоматически преобразует типы в некоторых случаях:

```javascript
console.log("5" + 3);   // "53" (конкатенация строк)
console.log("5" - 3);   // 2 (преобразование в число)
console.log("5" * "2"); // 10 (преобразование в число)
console.log("5" / "2"); // 2.5 (преобразование в число)
```

**Важно**: Неявное преобразование может приводить к неожиданным результатам, поэтому рекомендуется использовать явное преобразование.
