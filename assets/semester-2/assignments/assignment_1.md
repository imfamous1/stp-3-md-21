# Практическое задание 1: Веб-страница портфолио-резюме

## Описание задания

Создайте одностраничный веб-сайт — портфолио-резюме о себе. Используйте реальные актуальные данные, так как это задание поможет вам подготовить актуальное портфолио для поиска работы или профессионального развития.

## Технические требования

### Выбор технологий

**Для разработчиков:**
- Можно использовать любой стек технологий, который вам удобен или нравится

**Для начинающих:**
- Использовать стандартный стек: HTML, CSS, JavaScript

### Обязательное требование: использование нейросетей

Задание **обязательно** должно быть выполнено с использованием нейросетей. Допустимые инструменты:

- DeepSeek
- ChatGPT
- Perplexity
- Cursor
- Другие аналогичные инструменты

**Важно:** На сайте необходимо указать:
- Какую нейросеть вы использовали
- Сколько времени заняло выполнение задания

Эту информацию разместите в подвале (footer) сайта, например: `ChatGPT - 15 минут`

### Требование к JavaScript функционалу

Поскольку данный семестр по веб-дизайну посвящен языку JavaScript, на странице **обязательно** должен быть реализован функционал на JavaScript.

**Требование:** В подвале сайта укажите, какой функционал на JavaScript реализован (например: "Интерактивная форма обратной связи", "Анимация при прокрутке", "Динамическое изменение темы" и т.д.)

### Требования к верстке

- **Аккуратная и единообразная** верстка относительно макета
- **Системный подход** к дизайну
- **Современный внешний вид**
- Количество контента определяете самостоятельно

### Подход к разработке: Mobile First

**Обязательное требование:** Верстка должна быть выполнена по принципу **Mobile First**.

Если вы не знакомы с этим подходом, изучите его самостоятельно.

## Структура контента

Содержимое страницы должно включать:

- Информацию о себе
- Опыт работы (если есть)
- Образование
- Навыки
- Проекты (если есть)
- Контакты
- Другая релевантная информация

## Требования к оформлению

### Обязательные элементы на странице

1. **В подвале (footer) указать:**
   - Использованную нейросеть и время выполнения
   - Реализованный функционал на JavaScript

2. **Общий стиль:**
   - Современный дизайн
   - Адаптивная верстка (Mobile First)
   - Единообразие элементов

## Результат

### Формат сдачи

- Проект должен быть собран в **архив**
- Архив должен содержать все файлы проекта (HTML, CSS, JS, изображения и т.д.)

### Сроки сдачи

**До следующего четверга** (от всей группы сразу)

## Полезные советы

- Используйте реальные данные — это поможет вам создать актуальное портфолио
- Обратите внимание на типографику и цветовую схему
- Убедитесь, что сайт корректно отображается на мобильных устройствах
- Протестируйте интерактивные элементы на JavaScript
- Документируйте процесс работы с нейросетью

## Критерии оценки

- Качество верстки (аккуратность, единообразие)
- Соответствие принципу Mobile First
- Наличие и качество JavaScript функционала
- Современность дизайна
- Полнота информации о себе
- Указание использованной нейросети и времени выполнения
- Общее впечатление от сайта
